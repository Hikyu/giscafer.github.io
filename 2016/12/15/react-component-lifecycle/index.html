代码理解React组件生命周期过程

React Component整个生命周期分为三大块：初始化——存在期（更新）——销毁清理

首次实例化 调用方法

  • getDefaultProps()
  • getInitialState()
  • componentWillMount()
  • render()
  • componentDidMount()()

ES6写法中则为

  • constructor()
  • componentWillMount()
  • render()
  • componentDidMount()

存在期 调用方法

props 或 state状态改变

  • componentWillReceiveProps()
  • shouldComponentUpdate()
  • componentWillUpdate()
  • render()
  • componentDidUpdate()

销毁清理 调用方法

  • componentWillUnmount()
